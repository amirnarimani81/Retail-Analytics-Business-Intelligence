<h2>
Example SQL Analysis Query
</h2>



<p>

<b>Business Question:</b>

Which product categories generate the highest sales?

</p>



<pre>

SELECT 

category,

SUM(total_sale) AS total_sales,

COUNT(*) AS total_orders


FROM retail_sales


GROUP BY category


ORDER BY total_sales DESC;

</pre>



<p>

<b>Business Value:</b>

This analysis identifies the strongest product categories and supports inventory planning, marketing strategy, and revenue optimization.

</p>
